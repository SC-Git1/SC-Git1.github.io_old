Before we delve into the workshop, let us first familiarize ourselves with some terminologies that will be used. This glossary will serve as a handy reference throughout the workshop. Feel free to circle back to it anytime you need a refresher.

Glossary

TermDefinitionExample
Command Line Tool (CLI)A command line tool is a text interface to your computer. The CLI takes in commands, and passes them on to the shell. The shell interprets the command and then passes them on to the operating system to run.
Graphical User Interface (GUI)A graphical user interface is a visual way to interact with a computer using items like windows, icons, and menus.
KernelThe kernel is the heart of the OS. For example, Windows has the NT kernel, and macOS has the XNU kernel. NT kernel
(of Windows),
XNU kernel
(of macOS)
LinuxTechnically, Linux is a kernel. However, when most people say “Linux”, they are referring to a whole OS that uses the Linux kernel. There are many OS that use Linux, and they are called distributions. Ubuntu is one example of a Linux distribution.
Operating systemThe operating system is a software that serves as an intermediary between the user and the computer hardware. It allows users to interact with the computer without needing to understand the computer's language. It performs fundamental tasks like management of the file system, management of resources such as memory and processes, and handling input and output. Windows, macOS, Ubuntu
ShellA shell provides an interface between the user and the kernel of the OS. The shell executes programs called commands. bash
